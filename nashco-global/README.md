# Nashco Global

This is the official static website for Nashco Global, deployed using Cloudflare Pages.

## Structure
- All HTML files and assets are in the root directory for clean URLs.
- CSS: Assets/css/styles.css
- Images: Assets/images/
- PWA: manifest.json, sw.js

## Deployment
- Hosted on Cloudflare Pages with GitHub integration.
- Custom domain: nashcoglobal.com

## Editing
- Edit HTML, CSS, or images as needed.
- Push changes to GitHub to auto-deploy.
